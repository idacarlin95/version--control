Version control

git init
Creates a hidden git directory and enables Version control on the project
git clone "url"
Copies the git content of the specified URL into the current folder.
git pull
Updates the git project on the local machine by downloading it from origin.
git add "file"

Adds specfied files to the staging ground to prepare for a commit

Use "git add ." to add all files.

git commit -m "message"

Commits the file to branch on local machine (your own computer) and specifies a commit message.

Describe the things you have changed briefly in the message.

git push
Pushes the local commit to the branch on origin (the repository on github).
git checkout -b idabanch
Creates a new branch
